PassAuth

Making authentication more usable

We'll never share your email with anyone else.

Instructions:

1. Pick a username and password of your choice (there is no sign up process). Once you have picked up a username and password, continue using the same credentials to log in for all future instances.

2. Username and password fields have to be typed in everytime you log in. Only typed usernames and passwords will be accepted as valid. Saving passwords and using password managers will automatically be rejected- please refrain from using them on this website.
